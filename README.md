# Eagle Eye Security Services (EESS)

A modern, high-converting, single-page corporate landing page built for **Eagle Eye Security Services**, a premium security and facility management agency based in Moradabad, U.P. 

The website is designed to be highly professional, trustworthy, and authoritative—utilizing a navy blue and crimson red color palette with modern typography and sleek layout sections.

## ✨ Features

- **Modern & Responsive UI**: Fully responsive across desktop, tablet, and mobile devices.
- **Sticky Navigation**: A sticky header with smooth scrolling and a dedicated mobile hamburger menu.
- **Hero Section**: An engaging hero area with a high-quality overlay, dual-action buttons, and clear value propositions.
- **Service Highlights**: Clean grid layout utilizing `lucide-react` icons with subtle hover micro-animations.
- **Trusted Clients**: A section displaying top clients the agency serves.
- **Testimonials**: Clean, beautifully styled review cards with avatar placeholders and quotation background graphics.
- **Footer**: Comprehensive footer containing quick links and detailed contact information.

## 🛠️ Tech Stack

- **Framework**: [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Shlok1729/EESS.git
   cd EESS
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to the local URL provided in the terminal (usually `http://localhost:5173`).

## 📦 Build for Production

To create a production-ready build, run:
```bash
npm run build
```
This command bundles React in production mode and optimizes the build for the best performance. You can preview the production build using:
```bash
npm run preview
```
